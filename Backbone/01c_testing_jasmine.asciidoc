== Spies 
* Used by Jasmine to Mock functions
* created with +spyOn(Object, MethodName)+
* Matchers for Spies
** toHaveBeenCalled()
** toHaveBeenCalledWidth(Params)

== Spies Call Through 
* Call the Spy and the actual Function 


== Spies  Return
* set the Return Value of the Spy
.....
spyOn(foo, "Bar").andReturn(1234);
.....

== Spies Call Fake
* Redirect the call to a supplied function
.....
spyOn(foo, "Bar").andCallFake(function () { /* .... */});
.....

== Mocking the Clock
